Como Contribuir

Obrigado pelo interesse em contribuir com o OpenEduOps.

Nosso objetivo é construir software livre para resolver problemas operacionais de instituições educacionais e, ao mesmo tempo, criar oportunidades para aprendizado prático.

Quem Pode Contribuir

Qualquer pessoa.

Não importa se você:

Está estudando;
É recém-formado;
Trabalha na área;
Atua em outra profissão;
Possui anos de experiência.

Toda contribuição é bem-vinda.

Formas de Contribuir

Você pode contribuir com:

Código;
Documentação;
Testes;
Design;
UX/UI;
Infraestrutura;
Traduções futuras;
Revisão de código;
Sugestões de melhorias;
Relato de bugs.
Primeira Contribuição

Se esta é sua primeira participação em um projeto open source:

Leia a documentação;
Procure issues marcadas como:
primeira-contribuição
iniciante
boa-primeira-issue
Faça perguntas;
Não tenha receio de errar.

Aprender faz parte do processo.

Idioma

O idioma oficial do OpenEduOps é Português Brasileiro (pt-BR).

Issues, discussões e pull requests devem ser escritos prioritariamente em português.

Pull Requests

Ao abrir um pull request:

Explique claramente o problema;
Descreva a solução implementada;
Mantenha alterações pequenas e objetivas;
Respeite os padrões definidos pelo projeto.
Revisões

Revisões têm caráter educativo.

Nosso objetivo não é apenas aceitar código.

Nosso objetivo também é compartilhar conhecimento.

Dúvidas

Nenhuma dúvida honesta é considerada inadequada.

Pergunte.

Aprenda.

Contribua.

Ensine.
